# Progetto-7-Sasso-Carta-Forbici

Gioco interattivo Sasso Carta Forbice sviluppato con HTML, CSS e JavaScript.<br>
Il giocatore sfida il computer che genera una scelta casuale tramite Math.random().<br>
Il primo che raggiunge 3 punti vince la partita.<br><br>

Funzionalità<br>

Scelta tra Sasso, Carta e Forbice<br>

Generazione casuale della scelta del computer<br>

Sistema di punteggio per giocatore e macchina<br>

Messaggio dinamico del risultato<br>

Fine partita al raggiungimento di 3 punti<br>

Pulsante Reset per ricominciare il gioco<br><br>

Concetti JavaScript utilizzati<br>

- Funzioni<br>

- Oggetti (GAME_OPTIONS) <br>

- Math.random()<br>

- Manipolazione del DOM<br>

- Gestione eventi<br>

- Logica condizionale (if / else)<br><br>

Tecnologie<br>

- HTML<br>

- CSS<br>

- JavaScript
